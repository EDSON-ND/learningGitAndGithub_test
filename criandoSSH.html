Orientações para criar chave SSH no github

Abra o Git Bash;

Digite o comando:

"ssh-keygen -t id25519 -C 'digite o seu email'", aperte o ENTER

Informe a chave pública no site do github;

Para saber qual o valor da chave pública, navegue até a pasta;

Digite o seguinte comando "cat id_ed25519.pub"

Inicialize o SSH AGENT

digite o seguinte comando "eval $(ssh-agent -c)"

Adicione a chave privada no Git

Digite o seguinte comando "ssh-add id_ed25519"
